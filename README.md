
# Chess School Academy Website

This project is a web-based platform developed for a chess academy to manage courses, student registrations, and tournament updates. It provides a simple and user-friendly interface for students and parents to explore training programs, enroll in classes, and stay updated with academy activities.

The system includes role-based access for Admin, Students, and Coaches. The Admin can add or update course details, manage registrations, and post daily tournament updates. Students can register, view course information, and track updates easily.

The project is built using HTML, CSS, JavaScript, PHP, and MySQL. It focuses on simplicity, functionality, and real-world usability for small academy businesses.

## Features

* User Registration and Login
* Course Listing and Enrollment
* Tournament Updates Section
* Admin Dashboard for Content Management
* Responsive and Clean UI

## Purpose

This website was created to support a real chess academy business by improving online presence and simplifying management tasks.
